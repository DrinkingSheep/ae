: Will Co. Engine v2 :

- archives
arc -- full -- основной формат архивов. Все основные скрипты как правило содержатся в файле "Rio.arc".

- images
png -- std  -- основной формат изображений.
mos -- std  -- переименованные PNG файлы. Идут в паре с хентайными CG. Содержат маски для зацензуривания
               изображений для использования в галерее, где мозаика неэффективна ввиду сильного даунскейла.
pna -- no   -- сигнатура PNAP, содержит набор PNG изображений для отрисовки меню и системной графики.

- scripts
ws2 -- no   -- байт-кодовый скрипт, пошифрован циклическим сдвигом влево на 4 бита.

- sound
ogg -- std  -- *

- other
fx  -- no   -- файлы содержащие визуальные эффекты. (Возможно скомпилированные шейдеры?)

dat -- std  -- содержит видео со звуковой дорожкой. Возможные варианты:
               Контейнер: Windows Media Видео: VC-1 MP@HL Аудио: WMA Pro
               Контейнер: MPEG-PS       Видео: MPEG1      Аудио: mp2
lua -- no   -- скомпилированные lua скрипты
